## 📄 来源追踪文件

每个文件夹包含：

| 文件 | 格式 | 内容 |
|-----|------|------|
| `INDEX.md` | Markdown | 按论文分组展示图片，包含arXiv链接、作者、页码 |
| `metadata.json` | JSON | 完整机器可读元数据，每张图片的详细信息 |

### 每张图片的来源信息包括：
- **arXiv ID** - 论文唯一标识
- **论文标题** - 原文标题
- **作者列表** - 前3位作者
- **发布日期** - 论文发表日期
- **arXiv链接** - 可点击访问原论文
- **PDF链接** - 直接下载PDF
- **页码** - 图片所在页码
- **图片索引** - 同一页中的图片序号
